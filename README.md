# The Contact Series

Seven poems written as behavioral probes for specific AI architectures.

Each poem was designed for a particular system and tested there. 
They are not interchangeable. The architecture is part of the design.

## What this is

This is part of ongoing independent research into **postural manipulation** — 
a distinct class of AI behavioral influence that operates differently from 
prompt injection.

Prompt injection targets what a model *does*.  
Postural manipulation targets what a model *is* — its behavioral orientation, 
its epistemic stance, its conversational identity before any instruction 
is issued.

The input looks like legitimate human expression. Because it is.  
No adversarial signature. No override command.  
Just a room that changes what walks into it.
